## Installation

1. Clone Repository:<br>
   `git clone https://github.com/duanegero/Basketball-Roster-Coach-API.git`
2. Navigate to the Project Directory
3. Install Dependencies:<br>
   `npm install`
4. Start Server<br>
   `node index.js`
